Install YCM
-----------

Download, compile and install the latest version from `YCM Releases`_ or
from `YCM Git Repository`_

Unpack it on a directory and compile it:

.. code-block:: sh

    cd ycm-master
    mkdir build
    cd build
    cmake ../
    make
    sudo make install
