FindGLUT¶

Wrap kitware’s original FindGLUT script to work on windows with binary distribution. Standardize varibles.

In windows require you set GLUT_DIR

Set:

GLUT_FOUND
GLUT_LIBRARIES
GLUT_INCLUDE_DIRS
