FindGtkWin32 (Replaced by FindGTK2)Â¶

Warning

This module is deprecated. You should use FindGTK2 instead.

Searches and hopefully finds GTK on Windows.

Assumes that the environment variable GTK_BASEPATH is set to the place where GTK libs have been unpacked/installed. This is the default behavior of glade for win32, users that want to install gtk manually must define GTK_BASEPATH accordingly.

Creates:

GTK_INCLUDE_DIR   - Directories to include to use GTK
GTK_LIBRARIES     - Files to link against to use GTK
GTK_FOUND         - If false, don't try to use GTK
GTK_GL_FOUND      - If false, don't try to use GTK's GL features
